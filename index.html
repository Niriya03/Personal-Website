Cheat Sheets

HTML Cheat Sheet

  • <section> - Defines a section in a webpage, usually with related content, like a group of articles or a blog post.
  • <div> - A generic container element used to group content together, often used for styling or layout.
  • <h1> to <h6> - HTML heading tags, with <h1> as the main heading (largest) and <h6> as the smallest.
  • <img> - Used to embed images on a webpage, where the "src" attribute defines the image source, and "alt" provides alternative text.
  • <p> - Defines a paragraph of text.
  • <a href=""> - Creates hyperlinks to other web pages or external resources, with the "href" attribute specifying the destination URL.
  • <ul> - Creates an unordered list, typically with bullet points.
  • <ol> - Creates an ordered (numbered) list, where each list item is automatically numbered.
  • <li> - Represents a list item within an ordered or unordered list.
  • <table> - Defines a table for displaying tabular data, such as rows and columns.
  • <form> - Defines a form to collect user input, containing various form elements like text fields, checkboxes, etc.
  • <input> - Creates input fields in a form for users to enter data, such as text, numbers, or passwords.
  • <button> - Defines a clickable button that can trigger actions like form submission or navigation.
  • <iframe> - Embeds an external web page within the current page.
  • <video> - Embeds a video on the webpage, with the ability to play, pause, and control playback.

CSS Cheat Sheet

  • color: Sets the color of text content, allowing you to specify the color using names, hex, or RGB values.
  • background-color: Specifies the background color of an element, like a webpage or a container.
  • font-size: Defines the size of text. You can use units like pixels (px), em, rem, or percentages.
  • font-family: Specifies which font to apply to an element, enabling a wide range of typefaces and styles.
  • margin: Creates space outside an element, separating it from surrounding content.
  • padding: Adds space inside an element, between its content and its border.
  • border: Creates a boundary around an element, specifying width, style (solid, dashed), and color.
  • display: Defines the layout behavior of an element, such as block, inline, flex, or grid.
  • position: Positions an element relative to its normal position or another element, with values like static, relative, absolute, or fixed.
  • box-shadow: Adds a shadow effect to elements, with options for horizontal and vertical offset, blur radius, and color.
  • z-index: Controls the stacking order of elements. Higher values bring elements to the front, while lower values push them behind.
  • overflow: Manages content that overflows an element's box, with options like visible, hidden, or scroll.
  • height: Specifies the height of an element. This can be set in units like px, em, or percentages.
  • width: Specifies the width of an element. Like height, it can use px, em, percentages, and more.
  • hover: A pseudo-class that applies styles when a user hovers over an element with their mouse.

JavaScript Cheat Sheet

  • let/const: Declare variables in JavaScript. Use "let" for variables that can change and "const" for constants that shouldn't be reassigned.
  • document.querySelector: Selects the first element that matches a CSS selector, like an ID or class.
  • addEventListener: Attaches an event handler to an element, such as a click or keypress event.
  • forEach: A method that iterates over each element in an array, executing a function for each item.
  • if/else: Conditional statements that execute different code based on whether a condition is true or false.
  • function: Defines reusable blocks of code that can be called with arguments to perform specific tasks.
  • setTimeout: Executes a function after a specified delay (in milliseconds), commonly used for timed actions.
  • console.log: A method used to print messages to the console, mainly for debugging purposes.
  • innerHTML: Used to get or set the HTML content inside an element, such as a paragraph or div.
  • JSON.stringify/parse: Converts JavaScript objects to JSON format (stringify) and parses JSON strings into objects (parse).
  • parseInt/parseFloat: Functions that convert a string into an integer or floating-point number.
  • Array.prototype.map: Creates a new array by applying a function to each item in the original array.
  • setInterval: Calls a function repeatedly at specified intervals (in milliseconds), useful for animations or periodic tasks.
  • Math.random: Generates a pseudo-random number between 0 (inclusive) and 1 (exclusive), useful for random operations.
  • this: Refers to the current context of an object or function, often used within methods or event listeners.
